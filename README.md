
# AccessMate

A VS Code extension designed to improve code accessibility and readability for developers of all abilities.

## Features

- Enhanced syntax highlighting for better visual distinction
- Customizable font sizes and spacing options
- High contrast theme support
- Keyboard navigation optimization
- Screen reader compatibility
- Adjustable color palettes for color blindness support

## Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "AccessMate"
4. Click Install

## Usage

Access settings through VS Code preferences:
- `File > Preferences > Settings`
- Search for "AccessMate"
- Configure options to suit your needs

## Settings

- `accessmate.fontSize`: Adjust code editor font size
- `accessmate.lineHeight`: Customize line spacing
- `accessmate.contrastMode`: Enable high contrast mode
- `accessmate.colorBlindMode`: Select color blindness mode (deuteranopia, protanopia, tritanopia)

## Requirements

- VS Code 1.50.0 or higher
